§01 The Thread Count

Every tartan in the Scottish Register is stored as seven words. Not an image. Not a weave diagram. Not a swatch. B24 G4 B4 G4 B4 G24 R6 is the Black Watch. Thirty‑six characters encode three centuries of military identity: the dark blues of a government regiment, the greens of Highland forests, a single red stripe for the blood that distinguishes an armed unit from a civilian clan. That is the entire file format.

The trick is the pivot. The first and last threads in the sequence are reflection points: the pattern reads forward, hits the pivot, reads backward. This creates the bilateral symmetry that distinguishes tartan from gingham. No one called it a compression algorithm when they invented it. They called it a sett. But the structure is identical: a minimal description that expands into a full repeating tile. Thread count is source code. Everything else is rendering.

Thread count notation B24 G4 B4 G4 B4 G24 R6 Black Watch · Government No. 1

This explorer parses that notation and renders the result at pixel‑level accuracy: each thread crossing, each warp‑weft intersection, each color blend computed from the weave structure. The parser accepts all 4,200+ registered tartans in the Scottish Register format. Paste a thread count, get a tartan.

§02 The Palette

The register uses single‑letter codes for color. B is blue. R is red. K is black, because B was taken. Forty‑eight codes survive in the modern register, and each one maps to a specific dye that predates synthetic chemistry. Indigo came from woad. Red came from madder root. The expensive reds came from imported cochineal, which is why the most intricate red tartans belong to the wealthiest clans.

When I rendered all forty‑eight as a grid, something obvious appeared: the palette has structure. Dark blues and greens cluster together (government, hunting). Bright reds and whites cluster together (dress, ceremonial). The palette is not a box of crayons. It is a vocabulary with grammar. A tartan that mixes hunting greens with dress whites reads as wrong the same way a sentence with mismatched registers reads as wrong. The eye knows before the mind catches up.

§03 The Weaves

Same threads.
Six structures.
Six different tartans.

I assumed there was one way to weave a tartan. 2/2 twill: two‑over, two‑under, offset by one thread per row. The diagonal rib. The thing your brain files as "tartan." This is wrong. The Scottish Register acknowledges six weave structures. I built all six.

Plain weave is one‑over, one‑under. Flat, crisp, almost pixelated. Herringbone reverses the twill diagonal every few threads: a zigzag that creates more color blending at the reversal points. Hopsack pairs threads together, making the weave coarser. Satin floats one thread over four, creating a sheen. Broken twill interrupts the diagonal at irregular intervals. The result looks hand‑woven even on a screen.

The six specimens below all use the same thread count: Black Watch. Same colors, same stripe widths, same pivot structure. The only variable is the interlacing. Look at them and try to pick the one you would call "the real Black Watch." There is no correct answer. The register does not specify a weave.

Six weaves, one thread count, six tartans.
The pattern is not in the threads.
It is in the crossing.

§04 The Generator

Hit generate. The engine picks colors from the forty‑eight, assigns thread counts with weighted randomness, applies pivot symmetry, and renders the result through your chosen weave structure. Every pattern is mathematically valid. A loom could weave it. The Scottish Register could accept it, pending a clan petition and a two‑year review.

This is the question the tool eventually forces: at what point does generated plaid become tartan? The thread count notation is identical to a registered tartan's. The weave is authentic. The dye colors are period‑correct. Nothing is missing except three hundred years of someone wearing it.
